### 🔍 주요 기능

---

- **쿠폰 발급**: Redis의 싱글쓰레드를 활용하여 동시성 제어
- **결제 기능**: 토스페이먼츠 API를 활용한 결제 기능
- **영화 검색**: Elastic Search를 활용한 검색 기능
- **차트 기능**: 스케줄링과 캐싱을 활용한 차트 기능
